# 🍔 Food Ordering System (PHP + MySQL)

🚀 A full-stack food ordering web application inspired by platforms like Zomato and Swiggy.  
This system allows users to order food online, restaurants to manage menus, and admins to control the entire platform.


---

## 📌 Features

### 👤 User Panel
- User registration & login
- Browse restaurants & menus
- Add to cart & place orders
- View order history

### 🍽️ Restaurant Panel
- Restaurant registration & login
- Manage food items (Add/Edit/Delete)
- View & update orders

### 🛠️ Admin Panel
- Manage users & restaurants
- Monitor all orders
- Approve/reject restaurant registrations
- Full system control

---

## 🏗️ Architecture

This project follows a **Layered Architecture**:
- Presentation Layer (UI)
- Business Logic Layer
- Data Access Layer  

✅ Clean code  
✅ Easy maintenance  
✅ Scalable structure
